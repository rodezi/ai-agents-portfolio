# Description 

A simple conversational chatbot built using [LangGraph](https://www.langgraph.dev/) and Google's Gemini 2.0 Flash LLM.  
The chatbot supports multi-turn conversations and logs the entire dialogue to a `logging.txt` file for basic memory tracking.

## Features

- Uses **Google Gemini 2.0 Flash** as the language model
- Built on top of **LangGraph** for conversational state management
- Logs conversation history to `logging.txt`
- Simple terminal-based interface (or adaptable to any backend/API)
